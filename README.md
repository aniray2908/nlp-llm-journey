# NLP → LLM Journey

A self-paced learning repo documenting my path from NLP fundamentals
to building a mini GPT from scratch.

## Structure

| Folder | Topic |
|---|---|
| `00_pytorch_warmup` | Tensors, autograd, training loops |
| `01_nlp_fundamentals` | Tokenisation, embeddings, text classification |
| `02_demo_sentiment_analyser` | End-to-end NLP project |
| `03_transformer_deep_dive` | Attention from scratch, BERT fine-tuning |
| `04_llm_internals_finetuning` | Decoder models, LoRA, RAG |
| `05_capstone_mini_gpt` | Building a GPT from scratch |

## Stack
- Python 3.11+
- PyTorch
- HuggingFace Transformers
- Jupyter / Google Colab
